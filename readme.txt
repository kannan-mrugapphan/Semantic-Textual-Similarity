This project has two .py files - task2.py and task3.py
The task2.py file is used to extract features like pos tags, lemmas, hypernyms, etc
The task3.py file evaluates Semantic Textual Similarity between a pair of sentences and generates predictions.txt
predictions.txt is fed into evaluation.py to obtain the metrics of the model

The files are run by the following commands
python task2.py
python task3.py

###Have all the train, test and development files in the path where the project is unzipped###
